# 🚀 Synapse Agenda

**Advanced Collaborative Calendar & Workflow Manager**

Synapse is a personal full-stack project currently in **active development**. It is designed to be a unified ecosystem where your professional, academic, and personal lives coexist in perfect harmony through intelligent, role-based agendas.

🌐 **[Explore the Live App](https://synapse-eight-brown.vercel.app/)**

---

## ✨ Overview

Unlike standard calendars, Synapse understands that not all commitments are the same. It features specialized agenda types with unique business logic:

- **💼 Work**: Manage team events with a built-in approval workflow (Chiefs & Employees).
- **🎓 Education**: Dedicated views for Professors and Students to manage exams and assignments.
- **🏠 Personal**: Your private sanctuary for routines and appointments.
- **👥 Collaborative**: Share events with friends or family with one click.

## 🛠️ The Tech Stack (High Level)

The project is built using a modern, scalable architecture:

- **Frontend**: Built with **React** and **Vite** for a fluid, responsive user experience.
- **Backend**: A robust **Node.js** API using **Express** and **Prisma ORM**.
- **Real-time**: Powered by **Socket.io** for instant notifications and updates.
- **Database**: Professional-grade **PostgreSQL** hosted on Neon.
- **Cloud Infrastructure**: Distributed deployment across **Vercel** (Frontend) and **Render** (Backend).
